About

A flask app to download images from websites like - Idlebrain, Ragalahari, Cinejosh and Blogspot/sumon4all/tollywoodhq.com galleries by providing the gallery URL.

  • User enters a valid gallery URL.
  • The App downloads all the images from the given URL and saves them in a directory.
  • The directory is then Zipped and available for download
  • User can download the zipped directory on clicking 'Download' button.
Note: Only gallery URLs of the above mentioned websites will work.
